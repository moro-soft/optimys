# Оптимус v20 - Оптимизация размещения рулонов

Система оптимизации размещения рулонов в транспортных средствах (ТС) с 2D и 3D визуализацией.

## Описание

**Оптимус** — это веб-приложение для автоматической оптимизации загрузки рулонных грузов в различные типы транспортных средств:
- Контейнеры (20', 24', 40')
- Железнодорожные вагоны
- Еврофуры

Приложение использует алгоритмы гексагональной и жадной упаковки вокруг динамического центра для максимально эффективного использования пространства ТС.

## Основные возможности

### Версия 20 (index.htm - новая версия)
- Проверка перетаскивания мышью с разным поведением в 1D и 2D режимах
- Двойной клик - прилипание рулона к соседям
- Улучшенная работа с вариантами позиций
- Исправления ошибок переноса между контейнерами
- Исправления нарушений обжима контейнера при малом радиусе

### Версия 12.9
- Перенос рулона из ТС в ТС

### Версия 12.8
- Оптимизация сложных составов ТС
- Планирование схем размещения на цепочку ТС (контейнерный поезд)
- Автоматическое деление груза по транспортным средствам

### Ключевые функции

#### Типы транспортных средств
- Поддержка различных типов ТС: Авто, Вагон, Контейнер
- Настраиваемые параметры для каждого типа ТС

#### Алгоритмы оптимизации
- Гексагональная упаковка
- Жадная упаковка вокруг динамического центра
- Ветвление расстановки рулонов для минимизации дисбаланса
- Случайный выбор вариантов ветвления с закреплением для симметрии

#### Визуализация
- **2D визуализация** — схема вида сверху
- **3D визуализация** — трёхмерное представление загрузки
- Интерактивный сдвиг позиции рулона в обеих схемах
- Разноцветные группы рулонов

#### Управление схемой
- Перетаскивание рулонов мышью
- Сдвиг всех позиций продольно (X) ползунком
- `Shift` + сдвиг — перемещение всех рулонов правее
- `Ctrl` + сдвиг — перемещение всей стопки
- Центрирование нечётного крайнего рулона в верхнем ряду для баланса

#### Работа с данными
- Сохранение и загрузка схем размещения
- Скачивание и восстановление всех параметров
- Генерация случайных наборов рулонов
- Хранение указанных пользователем данных с автовосстановлением

#### Дополнительные функции
- Упоры как переворачивание стопки
- Адаптация для разных диаметров рулонов
- Поддержка разных диаметров в одном ТС
- Расчёт общего веса и дисбаланса
- Печатная форма схемы
- Варианты позиций для перетаскивания

## Технологии

- **HTML5/CSS3** — разметка и стили
- **JavaScript** — основная логика приложения
- **Three.js** — 3D визуализация
- **PHP** — серверная часть для сохранения/загрузки схем
- **XLSX** — работа с таблицами

## Установка и запуск

1. Разместите файлы проекта на веб-сервере с поддержкой PHP
2. Убедитесь, что папка `schemes/` имеет права на запись
3. Откройте `index.htm` в браузере (новая версия v20)
   - Или `index.html` для старой стабильной версии

## Требования к серверу

- Веб-сервер (Apache, Nginx)
- PHP 7.0 или выше
- Права на запись в директорию `schemes/`
